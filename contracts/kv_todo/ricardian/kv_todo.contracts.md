<h1 class="contract"> getbyaccname </h1>

Given an `account_name`, it will return back the associated entry in the TODO database.

<h1 class="contract"> upsert </h1>

Given a `uuid`, `account_name`, a `task`, and a `checked` predicated. This action will either insert a new record with the associated fields or update a preexisting one.

<h1 class="contract"> del </h1>

Given a `uuid` this action will delete the record associated to it.
